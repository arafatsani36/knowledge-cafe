Question

1. props and stats different.

Answer :
props:
1. Props are read-only.
2. Props are immutable.
3. Props are used to communicate between components.
4. Props make components reusable.

stats:
1. State changes can be asynchronous.
2. State is mutable.
3. States can be used for rendering dynamic changes with the component.
4. State cannot make components reusable.

2. What is the useState Hook.

Answer :
useState is React Hook that allows you to add state to a functional component. It returns an array with two values: the current state and a function to update it. The Hook takes an initial state value as an argument and returns an updated state value whenever the setter function is called.

4. How does react work.

Answer :
React basically maintains a tree for you. This tree is able to do efficient diff computations on the nodes. Think of your HTML code as a tree. React allows you to effectively re-construct your DOM in JavaScript and push only those changes to the DOM which have actually occurred.
